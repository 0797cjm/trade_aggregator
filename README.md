# Insulators Canada Trade Aggregator

Insulators Canada is a digital directory and trade hub for Heat and Frost Insulators in Canada. The live site is hosted at:

https://insulators.baymanbanter.com

This project is maintained as a BaymanBanter.com project.

## Purpose

The goal of this site is to give Canadian insulators a simple, centralized place to find union local information, dispatch links, trade resources, industry news, and public job board listings.

It is designed for workers who do not want to search across many different websites just to find local contact details, dispatch information, trade updates, or relevant job postings.

## Key Features

### Canadian Insulator Local Directory

The site lists Canadian Heat and Frost Insulator locals by province and region. Users can view local numbers, locations, contact details, website links, and directions.

### Local Finder

Users can select their province to find the most relevant insulator local. The selected local displays contact information, address, website, email, and a map.

### Live Dispatch Links

The site includes dispatch-related links for locals where available. Union members can quickly access official dispatch or work-call pages.

### Union News

The Union News section provides official union/local resource cards, including international union pages and selected local dispatch/update links.

### Trade Resources

The Trade Resources section provides stable official resource links for major industry topics such as LNG Canada, Newfoundland and Labrador energy resources, offshore regulators, and market snapshots.

### Trade News Stories

The site pulls trade-related news stories from topic-based news feeds. These stories focus on energy, offshore, LNG, industrial construction, and other work areas relevant to insulators.

### Public Job Board Listings

The job section aggregates public job listings from sources such as Job Bank and Careerjet. It prioritizes relevant insulator-related postings and filters out unrelated results where possible.

Current job features include:

- Job Bank RSS and HTML supplement parsing
- Careerjet API integration
- direct posting links where available
- source labels
- salary/pay display when available
- job highlight chips such as FIFO, Red Seal, Journeyperson, Apprentice, Camp, Travel, LOA, Remote, and Shutdown
- duplicate reduction and relevance filtering

### Light/Dark Theme

The site includes a light/dark visual switcher. The selected theme is stored locally in the browser.

## Current Data Sources

The site uses a mixture of official links, public feeds, and configured APIs.

Current sources include:

- Job Bank Canada
- Careerjet API
- selected union/local pages
- selected trade resource pages
- topic-based trade news feeds

Private API credentials are not stored in this repository.
